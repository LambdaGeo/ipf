# Apêndice: Real World Haskell (tradução PT-BR)

Uma tradução **não oficial** dos capítulos iniciais do livro *Real World Haskell*,
de Bryan O'Sullivan, Don Stewart e John Goerzen. Este material serve como leitura
complementar aos Módulos 1 e 2 da disciplina — vários trechos dos capítulos do
livro didático foram adaptados daqui.

## Capítulos traduzidos

* [Por que programação funcional? Por que Haskell?](por_que_haskell.md)

1. [Começando](cap01.md)
2. [Tipos e Funções](cap02.md)
3. [Definindo os tipos e entendendo as funções](cap03.md)
4. [Programação funcional](cap04.md)

## Demais capítulos (original em inglês)

Os capítulos não traduzidos podem ser lidos gratuitamente no site oficial do livro,
[book.realworldhaskell.org](http://book.realworldhaskell.org/read/). Destacam-se,
para esta disciplina:

* [Cap. 5 — Writing a library: working with JSON data](http://book.realworldhaskell.org/read/writing-a-library-working-with-json-data.html) (base do trabalho do Módulo 2)
* [Cap. 6 — Using typeclasses](http://book.realworldhaskell.org/read/using-typeclasses.html)
* [Cap. 7 — Input and output](http://book.realworldhaskell.org/read/io.html)
* [Cap. 11 — Testing and quality assurance](http://book.realworldhaskell.org/read/testing-and-quality-assurance.html) (base dos testes com QuickCheck)
